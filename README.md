U.S. States Guessing Game

An interactive Python game built using Turtle Graphics and Pandas, where players guess all 50 U.S. states and see them appear on the map in real time.

This project is part of Day 25 from Angela Yu's 100 Days of Code – Python Bootcamp.

📌 Features

🐢 Interactive map using Turtle Graphics

📊 CSV data handling using Pandas

🎯 Real-time score tracking

📝 Automatically generates states_to_learn.csv for missed states

🚪 Exit option with progress saving

🛠️ Built With

Python 3

Turtle (built-in module)

Pandas

cd us-states-game
2️⃣ Install dependencies
pip install pandas
3️⃣ Run the project
python main.py
🎮 How to Play

A blank U.S. map will appear.

Type the name of a state in the popup.

If correct, the state name appears on the map.

Type "Exit" anytime to quit.

The game will generate a file called:

states_to_learn.csv

This file contains the states you missed.
